

# 立体化学


## 立体化学的研究对象:构型, 构象

::: info

构成: 不同原子构成分子

构造: 相同的分子式有不同的构造方式

构型: 相同的构造有不同的构型

构象: 相同的构型有不同的构象

:::

eg: 如何确定一个分子: 3个C, 5个H, 1个O, 2个 Cl

 + 原子半径效应: $\ce{H-F<H-Cl<H-Br<H-I}$
