# 体表心电图

## 简介

常规心电图12导联体系：标准导联（双极肢体导联I、II、III）和单极导联（加压单极肢体导联（aVR、aVL、aVL）和胸壁导联（V1~V6））

双极和单极：

> 标准导联（双极肢体导联）：以右下肢体电极作为电学参照可以消除不必要的噪音，这样就存在了3对电极。在每一对电极中，其中一个电极作为导联的阳极端，电流流向这个电极时被描记成向上的方向（正向），而流向另一个电极时则描记成相反的波形，即标准导联（双极肢体导联）。
> + I导联：左上肢（\+）与右上肢（\-）之间的电位差（LA\-RA）
> + II导联：左下肢（\+）与右上肢（\-）之间的电位差（LL\-RA）
> + III导联：左下肢（\+）与左上肢（\-）之间的电位差（LL\-LA）
